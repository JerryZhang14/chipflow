# CHIPFLOW

A verified, trace-driven RV32I RISC-V microarchitecture platform.

## Goals

- Implement a five-stage pipelined RV32I processor in SystemVerilog.
- Verify its behavior against a Python reference model.
- Visualize instruction flow, hazards, forwarding, and performance from execution traces.
- Deploy to the DE10-Lite FPGA and report synthesis/timing results.
